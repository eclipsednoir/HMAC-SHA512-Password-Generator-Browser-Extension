How It Works

This Password Generator uses HMAC-SHA512 to generate a secure password based on the input provided by the user. The user needs to enter a "Password Base" and a "Secret Key". The combination of these inputs is hashed using the HMAC-SHA512 algorithm, and the resulting hash is encoded in Base64.

The generated password can be truncated to a desired length using the truncation options. The password can be copied to the clipboard for use.

Password Base
This is the base string for your password. It can be any string of characters that you can remember easily.

Secret Key
The secret key is a supplementary string used to enhance the security of your password using HMAC-SHA512.

Truncate Output
This option allows you to truncate the generated password to a desired length. The default length is 24 characters.

Select Truncate Length
This option allows you to select the desired length to which the password should be truncated.
